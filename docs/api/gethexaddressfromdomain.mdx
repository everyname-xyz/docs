# Get Name Space from Resolver Address
---

```bash
https://api.everyname.xyz:9010/search
```

This endpoint returns the domain name space for a specified payload query which contains the resolver address in hex format.
## Request Method

- POST

## Authentication

- API key through payload object

## Payload object

| Name   | Type   | Mandatory | Description                                                            | 
| ------ | ------ | --------- | ---------------------------------------------------------------------- |
| query | STRING | YES       | Resolver address in hex format. This address is responsible for managing domain records. |
| token | STRING | YES       | Your API token |

Payload object example: 

```json Response
{
    "query": "0xBF8AfA76BcC2f7Fee2F3b358571F426a698E5edD",
    "token": "YOUR_API_TOKEN"
}
```


## Returns

An object with the following fields:

- `domain`: (string) The domain name space connected to the given resolver smart contract address.

`````md codetabs
```go label="Go" title="main.go"
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://api.everyname.xyz:9010/search"
	payload := map[string]string{"query": "0xBF8AfA76BcC2f7Fee2F3b358571F426a698E5edD", "token": "YOUR_API_TOKEN"}

	payloadBytes, err := json.Marshal(payload)
	if err != nil {
		panic(err)
	}

	req, err := http.NewRequest("POST", url, bytes.NewBuffer(payloadBytes))
	if err != nil {
		panic(err)
	}

	client := &http.Client{}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	fmt.Println("response Status:", resp.Status)
	fmt.Println("response Headers:", resp.Header)
	body, _ := ioutil.ReadAll(resp.Body)
	fmt.Println("response Body:", string(body))
}
`````
