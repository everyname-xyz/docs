# Get Records for a Domain

---

```bash
https://resolve.everyname.xyz/domains/{domain}
```

This endpoint returns all the resolution records configured to a domain in a single response.

## Request Method

- GET

## Authentication

- Bearer Token/API Key

## URL Params

| Name   | Type   | Mandatory | Description                                                            |
| ------ | ------ | --------- | ---------------------------------------------------------------------- |
| domain | STRING | YES       | A domain name. See all the [supported domain endings](./supportedTLDS) |

## Returns

An object with the following fields:

- `address`: (string) The Resolver smart contract address. This contract is responsible for managing domain records.

`````md codetabs
```go label="Go" title="main.go"
package main


import (
  "fmt"
  "net/http"
  "io/ioutil"
)


func main() {


  url := "https://resolve.everyname.xyz/domains/:domainSearch"
  method := "GET"


  client := &http.Client {
  }
  req, err := http.NewRequest(method, url, nil)


  if err != nil {
    fmt.Println(err)
    return
  }
  req.Header.Add("Accept", "application/json")


  res, err := client.Do(req)
  if err != nil {
    fmt.Println(err)
    return
  }
  defer res.Body.Close()


  body, err := ioutil.ReadAll(res.Body)
  if err != nil {
    fmt.Println(err)
    return
  }
  fmt.Println(string(body))
}
`````
