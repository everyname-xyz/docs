# Get Records for a Domain

---

```bash
https://resolve.everyname.xyz/domains/{domain}
```

This endpoint returns all the resolution records configured to a domain in a single response.

## Request Method

- POST

## Authentication

- API key through payload object

## Payload object

| Name   | Type   | Mandatory | Description                                                            | 
| ------ | ------ | --------- | ---------------------------------------------------------------------- |
| query | STRING | YES       | A domain name + the domain ending. See all the [supported domain endings.](./supportedTLDS) |
| token | STRING | YES       | Your API token |

Payload object example: 

```json Response
{
    "query": "tom.eth",
    "token": "YOUR_API_TOKEN"
}
```


## Returns

An object with the following fields:

- `address`: (string) The Resolver smart contract address. This contract is responsible for managing domain records.

`````md codetabs
```go label="Go" title="main.go"
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://api.everyname.xyz:9010/search"
	payload := map[string]string{"query": "tom.eth", "token": "YOUR_API_TOKEN"}

	payloadBytes, err := json.Marshal(payload)
	if err != nil {
		panic(err)
	}

	req, err := http.NewRequest("POST", url, bytes.NewBuffer(payloadBytes))
	if err != nil {
		panic(err)
	}

	client := &http.Client{}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	fmt.Println("response Status:", resp.Status)
	fmt.Println("response Headers:", resp.Header)
	body, _ := ioutil.ReadAll(resp.Body)
	fmt.Println("response Body:", string(body))
}
`````
