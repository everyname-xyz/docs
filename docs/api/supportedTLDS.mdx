# Supported TLDS

---

We are constantly adding new TLDS to the API as we are expanding it. Currently the API supports the following name spaces:

- .eth,
- .sol
- .tez
- .algo
- .fn
- .dot
- .btc
- .bnb
- .stark
- .rsk
- .cosmos

We collect research and information regarding new blockchain domain name spaces which can be found [here.](https://github.com/everyname-xyz/Public)
