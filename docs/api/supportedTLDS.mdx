# Search Name

---

```bash
https://api.everyname.xyz:9010/tlds
```

This endpoint returns an array of the supported TLDS.

## Request Method

- GET

## Returns

Array response:

```json Response
[
  "algo",
  "apt",
  "avax",
  "bnb",
  "btc",
  "cosmos",
  "dot",
  "eth",
  "fn",
  "sol",
  "stark",
  "tez"
]
```
